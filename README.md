# Hyperlink-Redirect

A "useful" tool for turning hyperlinks into redirects in the name of shortening hyperlinks! 🧠 🧑‍🔬

## Run locally

### Install dependencies

    zef install --deps-only .

### Run the guy

    raku bin/hyperlink-redirect
